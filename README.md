# LA Software Cloud Remote

A simple cloud-based remote control system for the LA laptop alarm.

## Files

| File | Purpose |
|------|---------|
| `main.py` | FastAPI backend server |
| `requirements.txt` | Python dependencies |
| `remote.html` | Phone web page (host as index.html) |
| `DEPLOYMENT.md` | Step-by-step deployment guide |
| `MAC_INTEGRATION.md` | Swift integration guide |

## Quick Start

1. Deploy `main.py` to Render.com
2. Host `remote.html` on GitHub Pages
3. Update the BACKEND_URL in `remote.html`
4. Integrate with your Mac app using `MAC_INTEGRATION.md`
